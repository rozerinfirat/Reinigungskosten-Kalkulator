# Reinigungskosten-Kalkulator

Dieses Projekt ist ein einfacher Kalkulator für Reinigungsaufträge.

## Ziel
Der Kalkulator soll helfen, Reinigungskosten anhand von Arbeitszeit, Stundenlohn, Materialkosten und gewünschter Marge zu berechnen.

## Geplante Funktionen
- Eingabe der Quadratmeter
- Eingabe des Stundenlohns
- Berechnung der Arbeitskosten
- Berechnung der Materialkosten
- Berechnung eines Angebotspreises

Ich möchte meine Kenntnisse in Digitalisierung, Prozessoptimierung und später auch Programmierung praktisch anwenden.
